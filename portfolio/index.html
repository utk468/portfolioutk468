DIAMOND REGRESSION

TECH STACK

  • HTML
  • CSS
  • Flask
  • MACHINE LEARNING

Predicting diamond prices with machine learning showcases how AI can tackle complex real-world challenges. By cleaning the dataset, handling outliers with IQR, and encoding features like cut, color, and clarity, I streamlined data preprocessing using scikit-learn pipelines for efficient and consistent training.

Models like Linear Regression, Decision Trees, Random Forest, KNN, and XGBoost were evaluated, with XGBoost excelling in accuracy and minimal RMSE. Key features such as carat, clarity, depth, and dimensions enabled the model to capture critical patterns, while rigorous cross-validation ensured reliability.

Combining Python, scikit-learn, XGBoost, and Flask, this project integrates technical innovation with practical application. Explore the GitHub repository and detailed report for more insights into the process and its impact on pricing in luxury markets.

CANCER PREDICTION

TECH STACK

  • HTML
  • CSS
  • Flask
  • MACHINE LEARNING

This study explored a detailed cancer dataset with features such as Smoking, Passive Smoker, Genetic Risk, Alcohol Use, Occupational Hazards, Obesity, Air Pollution, and Chronic Lung Disease. Using bar plots, pie charts, histograms, heatmaps, and box plots, I gained valuable visual insights, handled outliers, and identified correlations to prepare the data effectively.

On the machine learning front, I implemented models like Random Forest, Decision Tree, XGBoost, and Support Vector Classifier (SVC) to predict cancer severity levels and analyze feature importance. The preprocessing pipeline included techniques such as label encoding, IQR-based outlier handling, and feature scaling. By using Python libraries like pandas, matplotlib, seaborn, scikit-learn, and XGBoost, I seamlessly combined robust data analysis with predictive modeling to uncover meaningful insights about cancer-related risk factors.

This project sharpened my expertise in data analysis, feature engineering, and model evaluation while deepening my understanding of cancer risk factors. Check out the GitHub repository for the code and detailed report for further insights. I’d love to hear your feedback or discuss the findings—feel free to connect or drop a comment!
